# Lanka Lottery 🎰 - Full-Stack Mobile Application

A secure, full-stack cross-platform mobile application designed for Sri Lankan users to purchase lottery tickets, manage a digital wallet, and view real-time transaction history. Built using a robust Express.js (TypeScript) backend and an interactive React Native (Expo) frontend.

---

## 🚀 Key Features
- **User Authentication:** Secure user signup and login powered by JWT (JSON Web Tokens) and Bcryptjs password hashing.
- **Wallet & Transactions:** In-app digital wallet for secure fund top-ups, ticket purchases, and comprehensive transaction logs.
- **Real-time Push Notifications:** Instant alert system for successful wallet operations and welcome broadcasts via Firebase Cloud Messaging (FCM) and Expo Notifications.

---

## 🛠️ Prerequisites
Before setting up the project, ensure you have the following installed on your local machine:
- **Node.js** (v18.x or higher)
- **MongoDB** (Local instance or MongoDB Atlas Cloud URI)
- **Expo Go App** (Installed on your physical Android or iOS device for testing)

Click the official Expo EAS build link below or scan the QR code from the build dashboard to download the `.apk` file:

### Step 2: Transfer to Phone (If downloaded via PC)
If you downloaded the file on your computer, transfer the `.apk` file to your Android device via USB Cable, WhatsApp Web, or Google Drive.

### Step 3: Enable Unknown Sources
Since this app is built for evaluation and not hosted on the Google Play Store, your phone might show a security warning. 
1. Open the downloaded `.apk` file on your phone.
2. If prompted, allow your browser or file manager to **"Install unknown apps"**.

### Step 4: Complete the Installation
1. Tap **Install**.
2. If a Google Play Protect popup appears saying *"Blocked by Play Protect"*, click on **"More details"** and select **"Install Anyway"**.
3. Once installed, open the **Lucky Lottery** app from your app drawer!
